# readme.txt

The true gaps for randomly selected subset of student answers in ScientsBank are contained in the file sel_tr_gaps.txt.

This folder also contains the file having gaps extracted in student answers (randomly selected from ScientsBank) using new triples (manually-generated triples for a pair of student-answer and model-answer). The folder also contains the file having gaps extracted in student answers (randomly selected from ScientsBank) using old triples (triples generated using ClausIE tool for a pair of student-answer and model-answer).
