The vetting-based strategy is employed for verification of annotated gaps in student answers belonging to all automatic short answer grading datasets UNT, ScientsBank, Beetle. The data for obtaining results of the strategy is created in the following manner:


1. Obtain the gaps annotated by the first annotator. 
2. The second annotator cross-verifies the gap annotations by marking against the gaps annotated by first annotator as 1, if agreeing with the first annotator annotated gap else 0 if disagreeing with the second annotator. 
3. An excel file "respective ASAG dataset"-rating.xlsx is created with all the above data. Such excel files are there in this folder.
